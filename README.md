# Basic Pricing Plan

This is a simple pricing plan layout that I created as a beginner web developer.
The goal of this project was to practice building clean layouts and improving my HTML and CSS skills.

features

* Responsive pricing cards
* Clean and simple design
* Beginner-friendly code structure

 Installation

```bash
git clone https://github.com/yourusername/basic-pricing-plan.git
cd basic-pricing-plan
```

Usage

Open the `index.html` file in your browser:

```bash
open index.html
```

Or just double-click the file to view it.

## 🛠️ Tech Stack

* HTML
* CSS
